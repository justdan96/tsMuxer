tsMuxeR – утилита для создания TS и M2TS файлов, для IP-вещания, а также для просмотра на аппаратных видеопроигрывателях (например, Dune HD Ultra, Sony Playstation 3, Samsung Smart TV и других).

tsMuxeR это программа с открытым исходным кодом.

Поддерживаемые форматы исходных файлов:

  • TS;
  • M2TS;
  • SIFF;
  • MOV;
  • MP4;
  • MKV;
  • Блюрей;
  • Возможность демукса.

Поддерживаемые видеокодеки:

  • H.264/MVC
  • H.265/HEVC;
  • Microsoft VC-1;
  • MPEG-2.

Поддерживаемые аудиокодеки:

  • AAC;
  • AC3 / E-AC3(DD+);
  • Dolby True HD (только для потоков с ядром AC3);
  • DTS/ DTS-HD;
  • LPCM.

Поддерживаемые типы субтитров:

  • M2TS Поток графической презентации (PGS).
  • SRT текстовые субтитры

Поддерживаемые контейнеры и форматы файлов:

  • Элементарный поток;
  • Транспортныйе потоки TS и M2TS;
  • Программные потоки EVO/VOB/MPG;
  • Матрёшка MKV/MKA.
  • MP4/MOV.

Основные свойства:

  • Поддержка 3D блюреев;
  • Поддержка UHD, HDR10, HDR10+ и Dolby Vision;
  • Автоматическая и ручная подстройка fps во время муксинга;
  • Изменение для H.264 значений: level, элементов SEI, SPS/PPS и блоков NAL а также их циклической вставки во время муксинга;
  • Временной сдвиг дорожек звука и субтитров относительно видео дорожки;
  • Возможность извлечения ядра DTS из DTS-HD;
  • Возможность извлечения ядра AC3 из True-HD;
  • Возможность склейки файлов;
  • Возможность подстройки fps для субтитров;
  • Возможность конвертирования LPCM потоков в WAVE и наоборот;
  • Вставка информации о языке дорожки в структуру блюрей и заголовок TS;
  • Возможность обрезать исходные файлы;
  • Возможность нарезать результирующие файлы;
  • Возможность определять задержки звука для исходных TS/M2TS/MPG/VOB/EVO;
  • Возможность удалять из видео дорожки информацию о pulldown;
  • Возможность использовать в качестве источника файлы плейлистов блюрея (MPLS);
  • Возможность конвертировать текстовые SRT субтитры в графические PGS (SUP);
  • Поддержка для SRT субтитров языка разметки для смены шрифтов, цвета, размера, и так далее; синтаксис этого языка подобен HTML;
  • Единый кроссплатформенный графический интерфейс для Windows, Linux, MacOS.
